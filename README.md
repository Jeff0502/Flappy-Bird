# Flappy-Bird
A flappy bird "clone" in C#, MonoGame.

Requirements:
=============

* Dotnet SDK version 7.0.400
* Windows 10 (Can possibly work in Windows 11)
* MonoGame.Framework.WindowsDX version 3.8.1.303
* MonoGame.Content.Builder.Task version 3.8.1.303
